Hamcrest-date

Date matchers for the Java Hamcrest library

Download this project as a .zip file Download this project as a tar.gz file

About

Hamcrest-date provides a selection of matchers compatible with @hamcrest for matching dates with test conditions. Some usage examples are below;

Examples

To test if a date is within 10 minutes of now then use:

MatcherAssert.assertThat(myDate, DateMatchers.within(10, TimeUnit.MINUTES, new Date()));

To test if a date is the same date as now then user:

MatcherAssert.assertThat(myDate,  DateMatchers.sameInstant(new Date()));
